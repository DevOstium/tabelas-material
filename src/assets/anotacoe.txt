		Alterar o caminho do git remote
	
	Antes
	$ git remote -v
	origin  http://192.168.10.43/internos/fenix/frontend/fenix-frontend.git (fetch)
	origin  http://192.168.10.43/internos/fenix/frontend/fenix-frontend.git (push)
